## 修复了以下内容

### “翼蚺”在“异界：沙漠”难以生成

主要原因是冰火传说在生成逻辑里硬编码了只能生成在草方块或丛林树叶上。

解决办法是如果原始方法返回 false 就进入 Hook 后的逻辑，如果脚下是沙子就返回 true

### 龙不掉落龙骨

原因是 ATM 使用了旧版冰火传说的数据包结构，而冰火传说社区版重写了这一结构并且改变了龙骨的物品 ID

解决办法是注册一个原始的数据包函数将逻辑重定向到新版数据包函数去，并且判断物品 ID 如果是旧版就替换为新版 ID

### 击杀冰火传说的”塞壬“后任务未完成

原因是因为 ATM-10 的击杀任务将实体 ID 错填为了元素魔艺（新生魔艺的扩展）的塞壬的实体 ID

解决办法是通过 Mixin 拦截任务判定逻辑。当代码发现在做一个“目标是元素魔艺塞壬、但图标却是冰火塞壬”的任务时，会自动帮你把击杀 冰火传说塞壬 也算作进度。
